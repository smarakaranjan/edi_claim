ISA (Interchange Header)
└─ GS (Functional Group)
   └─ ST (Transaction Set: 837)
      └─ BHT (Beginning of Hierarchical Transaction)
         └─ HL 1 (Billing Provider)
            ├─ NM1*85 (Billing Provider Details)
            ├─ NM1*87 (Pay-to Provider)
            ├─ HL 2 (Subscriber / Patient)
            │  ├─ SBR (Subscriber Info)
            │  ├─ NM1*IL (Patient)
            │  ├─ NM1*PR (Payer)
            │  ├─ HL 3 (Rendering / Service Provider)
            │  │  ├─ NM1*DN (Referring / Rendering Provider)
            │  │  ├─ LX (Service Line Counter)
            │  │  ├─ SV1 (Service Line: CPT / HCPCS, Charges)
            │  │  ├─ DTP (Service Date)
            │  │  ├─ REF (Reference IDs)
            │  │  └─ HI (Diagnosis Codes)
            │  └─ HL 4 (Optional Sub-loop: Additional Subscriber Service Line)
            │     ├─ SBR (Subscriber Info)
            │     ├─ NM1*IL (Patient)
            │     ├─ NM1*PR (Payer)
            │     ├─ HL 3 or 4 (Service Provider)
            │     │  ├─ LX (Service Line Counter)
            │     │  ├─ SV1 (Service Line)
            │     │  ├─ DTP (Service Date)
            │     │  └─ REF / HI (Reference / Diagnosis)
         └─ SE (Transaction Set Trailer)
      └─ GE (Functional Group Trailer)
└─ IEA (Interchange Trailer)
